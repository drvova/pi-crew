# Implementation Plan: pi-review Extension

## Overview

Create `/home/bom/source/my_pi/pi-review/` ‚Äî a standalone Pi extension providing multi-perspective code review, diff analysis, and review quality enforcement. The extension registers 3 tools (`review_diff`, `review_file`, `review_report`) and 5 slash commands (`/review`, `/review security`, `/review diff`, `/review report`, `/review file <path>`).

**Owner:** executor
**Dependencies:** Task 2
**Files to create:**

- `src/diff/context-extractor.ts` ‚Äî `extractContext(file, hunks, linesBefore=5, linesAfter=5)`: read file, extract surrounding context lines for each hunk
- `src/diff/impact-calculator.ts` ‚Äî `calculateImpact(hunks, cwd)`: count files changed, lines added/removed, identify high-impact files (large changes, critical paths like auth/config), produce `ImpactAssessment`
